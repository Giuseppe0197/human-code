Scomposizione esercizio pomeridiano

  • Aprire Slack
  • Leggere descrizione esercizio
  • Scaricare file zip
  • Aprire browser
  • Accedere a GitHub
  • Creare repo
  • Aprire Visual Studio Code
  • Copiare repo
  • Creazione file html e cartella css con file all'interno
  • Creazione cartella per immagini (se presenti)
  • Collegare file css a html
  • Aprire file html nel browser
  • Ricordarsi di salvare il file dopo ogni cambiamento
  • Verificare che il tutto funzioni con regole di debug
    • SE il tutto funziona
      • Procedere
    • ALTRIMENTI
      • Controllare se il link sia collegato correttamente
      • Controllare se ci sono errori di ortografia
      • Controllare se è stato dimenticato il punto e virgola alla fine
  • Impostare la pagina html suddividendola in varie sezioni con eventuali commenti
  • Divisione della pagina in blocchi con regole di debug
    • SE tutto combacia con la struttura desiderata
      • Iniziare la stesura dell'esercizio
    • ALTRIMENTI
      • Verificare e sistemare eventuali grandezze, margini, o disposizioni dei blocchi
  • Partire dal file html a scrivere contenuti o inserire immagini
  • Passare a file css per andare a modificare eventuali colori, dimensioni, layout, margini ecc.
  • Terminata la sezione fare un commit
  • Ripetere il procedimento per ogni sezione
  • SE insorgono problemi
    • Controllare di nuovo il codice aiutandosi anche con internet
  • ALTRIMENTI
    • Pushare e aprire un ticket, cercando di spiegare al meglio quale sia il problema
  • FINCHÉ non si ha risposta dal ticket
    • Continuare a cercare una soluzione al problema verificatosi
  • Quando il tutor risponde, spiegare nel dettaglio il problema incontrato
  • Trovare una soluzione
  • SE si capisce l'ostacolo e si supera
    • Continuare il lavoro passando alla prossima sezione (se terminata la precedente)
    • Commentare regole di debug (come il colore di sfondo) se non necessarie
  • ALTRIMENTI
    • Continuare a cercare la soluzione
    • OPPURE passare alla prossima sezione e ritornare in seguito sul problema
  • Alla fine di ogni sezione fare un commit
  • Terminata anche l'ultima sezione tornare su eventuali problemi irrisolti
  • SE tutto funziona
    • Pushare l'esercizio
  • ALTRIMENTI
    • Sistemare il file
  • Dopo il push continuare a migliorare il layout fino al completo soddisfacimento

Attraversare un incrocio

  • Camminare fino all'incrocio da attraversare
  • Controllare dove siano le strisce pedonali
  • Arrivare alle strisce
  • SE c'è il semaforo
    • Controllare il colore della luce
    • SE è rosso
      • Non attraversare
    • Aspettare che diventi verde
    • Al verde controllare che le macchine si siano fermate
    • Attraversare in tutta sicurezza
    • Non guardare il telefono mentre si attraversa
  • ALTRIMENTI
    • Controllare che ci siano segnali di stop
    • Guardare se stanno passando delle auto
    • Non attraversare senza controllare
    • SE ci sono delle auto che stanno passando
      • Non attraversare
      • Aspettare che una macchina si fermi
      • Quando le macchine sono ferme attraversare
    • ALTRIMENTI
      • Controllare scrupolosamente che non passino auto
      • Attraversare in sicurezza
      • Non distrarsi quando si attraversa

Scegliere un articolo sui social

  • Accendere il telefono
  • Scegliere l'applicazione da aprire
  • Aprire l'applicazione scelta
  • Scorrere la home
    • SE una notizia ci attira
      • Leggere l'eventuale descrizione
      • Leggere il titolo della notizia
      • Cliccare sul link
      • Leggere l'articolo
      • Uscire dal link una volta terminata la lettura
      • Mettere una reazione, condividere, commentare (se si vuole)
      • Taggare amici (se si vuole)
    • ALTRIMENTI
      • Continuare a scorrere la home fino alla notizia che interessa
      • Cercare la pagina dove si vuole vedere aggiornamenti recenti
      • Scegliere eventuali articoli, video ecc.
  • Andare sulle impostazioni
  • Uscire dall'applicazione
  • Bloccare il telefono
